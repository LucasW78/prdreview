# 说明文档

## 一、项目规划
- 项目名称：RAG 需求评审专家
- 目标：提供需求评审、知识库管理、问答检索、提示词管理与权限配置的一体化工作台。
- 当前阶段：功能迭代与稳定性优化阶段。

## 二、实施方案
- 前端：`React + TypeScript + Vite + TailwindCSS`，以组件化方式实现工作台与管理页面。
- 后端：`FastAPI + SQLAlchemy Async`，提供评审、知识库、权限等 API。
- 数据：关系库存储任务与配置，向量索引用于检索增强。
- 本次方案（提示词历史版本）：
  - 后端新增提示词历史版本持久化能力，存储于 `system_configs`。
  - 新增历史查询、版本回滚接口，保持与运行时提示词同步。
  - 前端提示词管理页新增历史版本列表、当前版本标识、回滚操作。
